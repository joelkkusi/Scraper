
# Scraper

A simple web scraper for [Book to Scrape](https://books.toscrape.com/)
## Run Locally

Clone the project

```bash
  git clone https://github.com/joelkkusi/Scraper
```

Go to the project directory

```bash
  cd Scraper
```

Install dependencies

```bash
  composer install
```

Run project from root

```bash
  php scraper.php
```
